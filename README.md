Custom Reaper Actions
Descr:
This repository is a collection of Python scripts, that automate repetitive actions I do in Reaper program. It serves the purpose of tracking my progress in
learning Python.
Scripts are written with the help of AI. Their functionality is tested in my reaper projects by me.

Scripts Overview

1. insert_track_circuit.py
Usage: Use it as an action from Actions / Show Action List / New Action / Load ReaScript 
Key Feature: Script creates track for purpose of instant recording from my circuit rhythm groovebox, on input 2.

2. region_creation_loop.py
Usage: Use it as an action from Actions / Show Action List / New Action / Load ReaScript
Key Feature: It creates region from time selection ,names it "A" and assigns random color to it. If "A" region exists, it names it "B" , all the way to the "Z".
